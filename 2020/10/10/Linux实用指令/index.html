Linux实用指令

1. 指令运行级别

七个运行级别：配置文件为/etc/inittab

  • 0：关机
  • 1：单用户【找回丢失密码】，在这个级别下进入linux系统不需要密码
  • 2：多用户状态没有网络服务
  • 3：多用户状态有网络服务【用的最多】
  • 4：系统未使用保留给用户【不要进入，未使用】
  • 5：图形界面
  • 6：系统重启

重用的运行级别是3和5，要修改默认的运行级别可以修改文件

/etc/inittab的id:5:initdefault:这一行的数字

切换指定运行级别的指令

  • init [0,1,2,3,5,6]

面试题

  • 如何找回root密码？（修改密码只能在主机进行，不能远程进行，所以还是很安全的）

    • 进入到单用户模式，然后修改root密码，因为进入单用户模式，root不需要密码就可以登录

    • 第一步，linux开机之前，按enter键进入此界面

    • 然后按e，进入到此界面

    • 选中第二个，然后按e进入到此页面

    • 然后输入空格，再输入1，进入单用户模式

    • 然后enter键，回到此界面，【告诉内核进入单用户模式】

    • 然后输入b，开始启动，进入单用户模式，而且以root模式登陆，然后输入passwd修改root密码，正常重启之后完成root密码修改

2. 帮助指令

当我们对某个指令不熟，可以使用linux提供的帮助指令来了解这个指令的使用方法

基本语法

  • man [命令或配置文件]

  • help 指令，获取shell内置命令的帮助信息

3. 文件目录指令

  • pwd 指令，显示当前工作目录的绝对路径

  • ls 指令，查看当前目录的所有内容信息

    • ls -a ，查看当前目录所有文件和目录，包括隐藏信息

    • ls -l ，以列表的方式显示信息

  • cd 指令，切换到指定目录

    • cd [参数]
    • cd ~：回到自己的家目录，普通用户家目录在/home下，root用户家目录在/root
    • cd ..：回到当前目录的上一级目录也可以直接用cd命令，更快捷

  • mkdir 指令：创建目录

    • mkdir 目录

    • mkdir -p 目录名：创建多级目录

  • rmdir 指令：删除空目录

    • rmdir 目录

    • rm -rf 要删除的内容：删除非空目录

    • rm -rf /*：删库跑路，慎重选择

  • touch 指令：创建空文件

    • touch 文件名称

  • cp指令，拷贝文件到指定目录

    • cp [选项] source dest：将source文件拷贝到dest，将world.txt复制到bbb文件夹

    • -r：递归复制整个文件夹 ：cp -r resource dest

    • 强制覆盖不提示：\cp

  • rm指令：移除或删除文件或目录

    • rm [选项] 要删除的文件或目录

    • -r：递归删除整个文件夹，-f：强制删除不提示

    • rm -rf /*：删除所有文件，且不提示

  • mv指令：移动文件与目录重命名

    • mv oldNameFile newNameFile （重命名）

    • mv /temp/movefile /targetFolder（移动文件）

  • cat 指令：查看文件内容，只是以只读的方式打开

    • cat [选项] 要查看的文件

    • -n：显示行号

    • cat只能浏览文件，不能修改文件，为了浏览方便，一般会带上管道命令|more

  • more指令

    • more指令是一个基于VI编辑器的文本过滤器，它以全屏幕的方式按页来显示文本文件的内容，more指令中内置了若干快捷键

    • 快捷键：

      操作 功能说明
      空白键(space) 代表向下翻一页；
      Enter 代表向下翻『一行』；
      q 代表立刻离开 more ，不再显示该文件内容。
      Ctrl+F 向下滚动一屏
      Ctrl+B 返回上一屏
      = 输出当前行的行号
      :f 输出文件名和当前行的行号

  • less指令

    • less指令用来分屏查看文件内容，它的功能与more指令类似，但是比more指令更加强大，支持各种显示终端。less指令在显示文件内容时，并不是一次将整个文件加载之后才显示，而是 根据显示需要加载内容，对于显示大型文件具有较高的效率。

    • 快捷键

      操作 功能说明
      空白键(space) 代表向下翻一页；
      Enter 代表向下翻『一行』；
      q 代表立刻离开 less ，不再显示该文件内容。
      pagedown 向下翻动一页
      pageup 向上翻动一页
      /字串 向下搜寻字串的内容
      ？字串 向上搜寻字串的内容

  • 输出重定向指令 > 和追加指令 >>

    • 输出重定向>：会将原来的文件的内容覆盖掉

      • 例如 ls -l > 文件 （将列表的内容写入到文件中(覆盖写)）
      • 例如ls >> 文件（将列表的内容追加到文件中）
      • 例如cat 文件1>>文件2（将文件1的内容追加到文件2）

  • echo指令：输出内容到控制台

    • echo [选项] [输出内容]

  • head指令

    • 用于显示文件的开头部分，默认情况下head指令显示文件的前十行内容

    • head -n 5 文件：显示文件头五行内容

  • tail指令

    • 用于输出文件中尾部内容，默认输出文件的后十行

    • tail -n 5 文件：查看文件后五行的内容

    • tail -f 文件：实时追踪该文档的所有更新，工作中经常使用，ctrl+c退出tail模式

  • ln指令

    • 软连接也叫符号链接，类似于windows中的快捷方式，主要存放了链接其他文件的路径

    • ln -s [原文件目录] [软链接名]：给原文件创建一个软连接

    • 此时，进入到linkToRoot文件夹，显示的内容是root文件夹中的内容，但是此时你输入pwd，显示的还是本路径文件夹

    • 删除软链接

    • 在删除软链接时，不要带有/，否则提示资源忙

  • history命令

    • 查看已经执行过的历史命令，也可以执行历史命令

    • 显示最近使用过的10个指令，执行编号为149的历史指令

4. 时间日期指令

  • date指令：显示当前日期

    • date：显示当前时间

    • date +%Y：显示当前年

    • date +%m：显示当前是哪个月

    • date +%d：显示当前是那一天

    • date “+%Y-%m-%d %H:%M:%S”：显示年月日时分秒

  • date指令-设置日期

    • date -s 字符串时间

    • cal[选项]：查看日历指令，不加选项，显示本月日历

    • cal 2020 ：显示2020年日历

5. 搜索查找指令

  • find指令：find指令将从指定目录向下递归地遍历其各个子目录，将满足条件的文件或者目录显示在终 端。

    • find [搜索范围] [选项]

    • 选项说明

      选项 功能
      -name<查询方式> 按照指定的文件名查找模式来查找文件
      -user<用户名> 查找属于指定用户名所有文件
      -size<文件大小> 按照指定的文件大小查找文件

    • 按文件名：根据名称查找/home目录下的hello.txt文件

    • 按拥有者：查找/opt目录下，用户名称称为root的文件

    • 查找整个linux系统下大于20m的文件（+n 大于 -n小于 n等与）

    • 查询 /目录下所有txt文件

  • locate指令

    • locaate指令可以快速定位文件路径。locate指令利用事先建立的系统中所有文件名称及路径 的locate数据库实现快速定位给定的文件。Locate指令无需遍历整个文件系统，查询速度较快。 为了保证查询结果的准确度，管理员必须定期更新locate时刻。

    • 由于locate指令基于数据库进行查询，所以第一次运行前，必须使用updatedb指令创建locate数据库

  • grep指令和管道符号|

    • grep过滤查找，管道符”|”表示将前一个命令的处理结果输出传递到后面的命令处理

    • grep [选项] 查找内容 源文件，-n：显示匹配行及行号，-i：忽略大小写

    • 案例：查找a.txt文件中，”ces”所在的行，并显示行号

    • grep -ni ces：表示不区分大小写

6. 压缩和解压文件命令

  • gzip/gunzip 指令

    • gzip 文件：用于压缩文件，只能将文件压缩为*.gz文件

    • gunzip 文件.gz：用于解压文件

    • 当我们使用gzip对文件进行压缩后，不会保留原来的文件，解压后也不会保留压缩包

  • zip/unzip 指令

    • zip [选项] 自定义文件名.zip ：压缩文件和目录的命令

      • -r：递归压缩：即压缩目录

    • unzip [选项] 文件名.zip ：解压缩文件

    • 常用选项

      • -d<目录> ：指定解压后文件的存放目录

  • tar指令

    • tar指令是打包指令，最后打包后的文件是.tar.gz的文件

    • tar [选项] XXX.tar.gz 打包的内容 (功能描述：压缩后的文件格式.tar.gz)

    • 选项说明

      选项 功能
      -c 产生.tar打包文件
      -v 显示详细信息
      -f 指定压缩后的文件名
      -z 打包的同时压缩
      -x 解包.tar文件

    • 案例：压缩多个文件，a.tar.gz是打包后的文件名

    • 案例：解压test01文件夹下，test01文件夹要事先存在才能成功
