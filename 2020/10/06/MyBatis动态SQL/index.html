MyBatis动态SQL

1.1 概述

动态 SQL是MyBatis强大特性之一。极大的简化我们拼装 SQL的操作。

动态 SQL 元素和使用 JSTL 或其他类似基于 XML 的文本处理器相似。

MyBatis 采用功能强大的基于 OGNL 的表达式来简化操作。

  • if
  • choose (when, otherwise)
  • trim (where, set)
  • foreach
